    **base.html changes:**

    2. Add `data-state` attribute to the `<body>` tag:
       Change the existing `<body style="...">` to:
       ```html
       <body data-state="{{ state_code|default('') }}" style="...existing styles...">
       ```

    3. Add the inline alert banner script BEFORE the closing `</body>` tag (after the language switcher include, before `</body>`). The script must be under 2,500 bytes UTF-8.

       The script should:
       - Read `data-state` from `document.body`
       - Build URL: `/api/alerts` + (state ? `?state=${state}` : "")
       - Fetch with a 5-second timeout (using AbortSignal.timeout if available, else setTimeout+abort fallback)
       - On success with alerts: create a container div, style it with inline styles, create a paragraph per alert with severity color, headline, and source link
       - Add "Alerts as of: [time]" timestamp line at bottom
       - Insert the container as the first child of `<body>` (above everything including skip link)
       - On failure or empty alerts: do nothing (no DOM changes, no console errors)
       - Use `.catch(function(){})` to swallow all errors silently

       **Severity color mapping (inline in script):**
       - Extreme: #d32f2f (red background tint)
       - Severe: #e65100 (dark orange)
       - Moderate: #f57f17 (amber)
       - Minor: #1565c0 (blue)
       - Unknown: #616161 (grey)

       **Banner styling (all inline, no CSS classes):**
       - Container: `border-bottom:2px solid #b71c1c; padding:0.5rem 0.75rem; font-size:0.85rem; max-width:45em; margin:0 auto;`
       - Background: use a light tint based on highest severity alert (e.g., #ffebee for Extreme/Severe, #fff3e0 for Moderate, #e3f2fd for Minor)
       - Each alert line: `margin:0.25rem 0;` with colored severity label (bold, font-weight:bold) + headline text + "source" link
       - Timestamp line: `font-size:0.75rem; color:#616161; margin:0.25rem 0;`

       **Script size target:** Under 1,000 bytes minified. Use short variable names, no whitespace. Write it readable first, then minify.

       Example readable version:
       ```javascript
       (function(){
         var s=document.body.getAttribute("data-state")||"";
         var u="/api/alerts"+(s?"?state="+s:"");
         fetch(u).then(function(r){return r.json()}).then(function(d){
           if(!d.alerts||!d.alerts.length)return;
           var c=document.createElement("div");
           var sv={"Extreme":"#d32f2f","Severe":"#e65100","Moderate":"#f57f17","Minor":"#1565c0","Unknown":"#616161"};
           var bg={"Extreme":"#ffebee","Severe":"#fff3e0","Moderate":"#fff8e1","Minor":"#e3f2fd","Unknown":"#f5f5f5"};
           c.style.cssText="border-bottom:2px solid #b71c1c;padding:.5rem .75rem;font-size:.85rem;max-width:45em;margin:0 auto;background:"+(bg[d.alerts[0].severity]||"#fff3e0");
           d.alerts.forEach(function(a){
             var p=document.createElement("p");
             p.style.cssText="margin:.25rem 0";
             p.innerHTML='<b style="color:'+(sv[a.severity]||"#333")+'">'+a.severity+'</b> '+a.headline+(a.link?' <a href="'+a.link+'" style="color:#1565c0;font-size:.8rem">source</a>':"");
             c.appendChild(p)
           });
           if(d.cached_at){
             var t=document.createElement("p");
             t.style.cssText="margin:.25rem 0;font-size:.75rem;color:#616161";
             t.textContent="Alerts as of: "+new Date(d.cached_at).toLocaleString();
             c.appendChild(t)
           }
           document.body.insertBefore(c,document.body.firstChild)
         }).catch(function(){})
       })();
       ```

       Minify this to a single line in the template. The `<script>` tag wrapping it should look like:
       ```html
       <script>!function(){var s=document.body.getAttribute("data-state")||"",u="/api/alerts"+(s?"?state="+s:"");fetch(u).then(function(r){return r.json()}).then(function(d){if(!d.alerts||!d.alerts.length)return;var c=document.createElement("div"),v={"Extreme":"#d32f2f","Severe":"#e65100","Moderate":"#f57f17","Minor":"#1565c0","Unknown":"#616161"},b={"Extreme":"#ffebee","Severe":"#fff3e0","Moderate":"#fff8e1","Minor":"#e3f2fd","Unknown":"#f5f5f5"};c.style.cssText="border-bottom:2px solid #b71c1c;padding:.5rem .75rem;font-size:.85rem;max-width:45em;margin:0 auto;background:"+(b[d.alerts[0].severity]||"#fff3e0");d.alerts.forEach(function(a){var p=document.createElement("p");p.style.cssText="margin:.25rem 0";p.innerHTML='<b style="color:'+(v[a.severity]||"#333")+'">'+a.severity+"</b> "+a.headline+(a.link?' <a href="'+a.link+'" style="color:#1565c0;font-size:.8rem">source</a>':"");c.appendChild(p)});if(d.cached_at){var t=document.createElement("p");t.style.cssText="margin:.25rem 0;font-size:.75rem;color:#616161";t.textContent="Alerts as of: "+new Date(d.cached_at).toLocaleString();c.appendChild(t)}document.body.insertBefore(c,document.body.firstChild)}).catch(function(){})}();</script>
       ```
