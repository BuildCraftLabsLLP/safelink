Offline indicator (the full minified script from RESEARCH.md):
```html
<script>!function(){function u(){var e=document.getElementById('_ol');if(e)e.style.display=navigator.onLine?'none':'block';}window.addEventListener('online',u);window.addEventListener('offline',u);if(!navigator.onLine){var e=document.createElement('p');e.id='_ol';e.setAttribute('style','background:#b71c1c;color:#fff;text-align:center;padding:.25rem;font-size:.85rem;margin:0');e.textContent='You are offline \u2014 information may be outdated';document.body.insertBefore(e,document.body.firstChild);}window.addEventListener('offline',function(){if(!document.getElementById('_ol')){var e=document.createElement('p');e.id='_ol';e.setAttribute('style','background:#b71c1c;color:#fff;text-align:center;padding:.25rem;font-size:.85rem;margin:0');e.textContent='You are offline \u2014 information may be outdated';document.body.insertBefore(e,document.body.firstChild);}});u();}();</script>
```

**Area 3: Generate sitemap.xml after all pages are built.**
First, add `import xml.etree.ElementTree as ET` to the imports section at the top of build.py (it is not currently imported).

Then add a new function `generate_sitemap(data)` that:
- Uses `xml.etree.ElementTree` (imported as ET above)
- Base URL: `https://safelink.serverlord.in`
- Creates `<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">`
- For each language in LANGUAGES, for each page type, generate the URL:
  - Homepage: `/{lang_prefix}/` (English: `/`, others: `/hi/` etc.)
  - State pages: `/{lang_prefix}/state/{slug}/`
  - District pages: `/{lang_prefix}/state/{state_slug}/district/{dist_slug}/`
  - City pages: `/{lang_prefix}/state/{state_slug}/city/{city_slug}/`
  - Guide pages: `/{lang_prefix}/guide/{guide_type}/`
  - Static pages: `/{lang_prefix}/{page_name}/` for each content page name (about, disclaimer, firstaid, kit, recovery)
- Only `<loc>` tags (no lastmod, priority, changefreq — keeps file smaller)
- Write to `DIST_DIR / "sitemap.xml"` using `tree.write()` with `xml_declaration=True, encoding='unicode'`
- Print: `Generated sitemap.xml with {count} URLs`

Call `generate_sitemap(data)` in `build()` after all pages are built (after the for lang loop, before build summary).

**Area 4: Copy static/ files to dist/ during build.**
Add a new function `copy_static_files()` that:
- Source dir: `BASE_DIR / "static"`
- If static dir doesn't exist, print warning and skip
- Copy each file in static/: `manifest.json`, `sw.js`, `icon-192.png`, `icon-512.png`, `robots.txt`
- Use `shutil.copy2(src, DIST_DIR / filename)` for each
- Print: `Copied {n} static files to dist/`

Call `copy_static_files()` in `build()` after DIST_DIR is created and before page generation starts (so icons are available, though they're not referenced during HTML generation).

**deploy.sh change:**
In the `cat > dist/_headers` heredoc, add the sw.js no-cache rule. The current heredoc only has `/*` rules. Add AFTER the `/*` block (but before HEADERS terminator):

```
/sw.js
  Cache-Control: no-cache, no-store, must-revalidate

The final _headers content should look like:

/*

  Referrer-Policy: strict-origin-when-cross-origin
  Cache-Control: public, max-age=3600

/sw.js
  Cache-Control: no-cache, no-store, must-revalidate
```
